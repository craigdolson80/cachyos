# =====================================================
# NIRI KEYBINDINGS HELP
# =====================================================
# Modifier legend:
#   Super  = ⌘ / Windows key
#   Alt    = Alt / Option
#   Ctrl   = Control
#   Shift  = Shift
#
# Notes:
# - This file is user-maintained
# - Order reflects muscle memory, not categories
# - Keep descriptions short (≤ 40 chars)
# =====================================================

# ----------------------
# Core / Session
# ----------------------
Super + Enter        Launch terminal
Super + Q            Close focused window

# ----------------------
# Launchers
# ----------------------
Super + Space        App launcher (fuzzel)

# ----------------------
# Window Focus
# ----------------------
Super + Left            Focus left
Super + PageDown        Focus Workspace Down
Super + PageUp          Focus Workspace Up
Super + Right           Focus right
Super + Shift + Left    Focus Monitor left
Super + Shift + Right   Focus Monitor right
Super + Tab             Cycle windows
Alt + Tab               Cycle windows

# ----------------------
# Window Movement
# ----------------------
Super + Control + Right     Move Column Right
Super + Control + Left      Move Column Left
Super + Shift + PageUp      Move Workspace Up
Super + Shift + PageDown    Move Workspace Down
Super + Control + Shift + Left      Move To Monitor Left
Super + Control + Shift + Right     Move To Monitor Right


# ----------------------
# Layout / Tiling
# ----------------------
Super + Space        Toggle floating
Super + F            Toggle fullscreen
Super + V            Toggle split

# ----------------------
# Workspaces
# ----------------------
Super + 1            Workspace 1
Super + 2            Workspace 2
Super + 3            Workspace 3
Super + 4            Workspace 4
Super + 5            Workspace 5

Super + Control + 1    Move window → WS 1
Super + Control + 2    Move window → WS 2
Super + Control + 3    Move window → WS 3
Super + Control + 4    Move window → WS 4
Super + Control + 5    Move window → WS 5

# ----------------------
# Window Sizing
# ----------------------
Super + +   Increase Window Size
Super + -   Decrease Window Size
Super + F   Set Window to Full Screen

Super + Alt + L      Lock screen
